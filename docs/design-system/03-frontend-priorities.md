# P0/P1/P2/P3 — Приоритеты по страницам

## Главная формула
```
P0 — критическое (всегда видно сразу)
P1 — частое (видно без глубокого поиска)
P2 — дополнительное (вкладки, аккордеоны, drawer)
P3 — редкое (убирается глубже, но не исчезает)
```
